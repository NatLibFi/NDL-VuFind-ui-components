---
title: Alert Success
---

#### Description

An alert implying that users request was successful. Can be used e.g. in forms.

#### Properties

| Prop        | Type      | Value           | Required |
| ----------- | --------- | --------------- | -------- |
| alert       | `String`  | -               | Yes      |
| alertClass  | `String`  | `alert-success` | Yes      |
| closeLabel  | `String`  | -               | No       |
| dismissable | `Boolean` | -               | No       |
