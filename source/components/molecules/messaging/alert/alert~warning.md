---
title: Alert Warning
---

#### Description

An alert used for warning users that certain actions can cause unexpected behaviour.

#### Properties

| Prop        | Type      | Value           | Required |
| ----------- | --------- | --------------- | -------- |
| alert       | `String`  | -               | Yes      |
| alertClass  | `String`  | `alert-warning` | Yes      |
| closeLabel  | `String`  | -               | No       |
| dismissable | `Boolean` | `true`          | No       |
