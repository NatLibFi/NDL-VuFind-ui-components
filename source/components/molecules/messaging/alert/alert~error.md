---
title: Alert Error
---

#### Description

An alert implying that something went wrong. Can be used e.g. in forms.

#### Properties

| Prop        | Type      | Value          | Required |
| ----------- | --------- | -------------- | -------- |
| alert       | `String`  | -              | Yes      |
| alertClass  | `String`  | `alert-danger` | Yes      |
| closeLabel  | `String`  | -              | No       |
| dismissable | `Boolean` | -              | No       |
