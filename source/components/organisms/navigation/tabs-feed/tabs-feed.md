#### Description

Tabs component with content loaded dynamically from feed.

#### Rendering

**Rendering with a helper function is supported:**

```
<?= $this->feedTabs(['title' => 'Tabs feed with heading', 'ids' => ['Feed tab 1' => 'tab1', 'Feed tab 2' => 'tab2', 'Feed tab 3' => 'tab3']]) ?>
```
