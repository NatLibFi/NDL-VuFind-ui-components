#### Description

Record tabs component, with the last tab aligned to right. Tab contents are dynamically loaded from a feed. Used on record landing pages.

#### Properties

| Prop         | Type       | Required |
| ------------ | ---------- | -------- |
| menuClass    | `String`   | No       |
| tablistLabel | `String`   | Yes      |
| tabs         | `Object[]` | Yes      |
